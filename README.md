laserwave.vim
=============

[![MIT License](http://img.shields.io/badge/license-MIT-blue.svg?style=flat)](LICENSE)

An clean 80's synthwave / outrun inspired theme for Vim.


## Installation
### Use dein.vim
```vim
call dein#add('AjxLab/laserwave.vim'')
```
### Use vim-plug
```vim
Plug 'AjxLab/laserwave.vim'
```
